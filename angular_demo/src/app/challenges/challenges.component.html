Angular Challenge Exercises

  1. Create a new component: Use ng generate component my-demo and display a message.
  2. Build a custom pipe: Transform text to uppercase.
  3. Make a service: Share data between two components.
  4. Add a new route: Create a page and add it to the menu.
  5. Connect to an API: Fetch and display data from /api/products.
  6. Write a unit test: Test a function in your app.
  7. Accessibility challenge: Make a form fully accessible.
